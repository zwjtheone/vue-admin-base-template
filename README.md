### 启动说明

``` bash
# 进入项目根目录
cd **

# 安装依赖文件
npm install

# 启动本地开发环境
npm run start

# 发布开发环境
npm run dev

# 发布测试环境
npm run test

# 发布生产环境
npm run pro

# 使用mock数据发布
npm run mock
```
